# Game of Cars 🏎️

Un simulador de conducción sofisticado en 2D programado con HTML5 Canvas y JavaScript Vanilla (ES6+). Implementa físicas realistas de vectores, fricción, aceleración y un sistema de partículas dinámico para simular el humo de los neumáticos al derrapar.

## Características
* **Motor de Físicas Avanzado:** Movimiento vectorial basado en ángulos reales de giro.
* **Sistema de Partículas:** Humo reactivo al derrapar a altas velocidades.
* **Manejo de Colisiones:** Sistema perimetral que detecta los bordes de la pista e impide que el vehículo atraviese los muros.
* **Cero Dependencias:** Construído exclusivamente con tecnologías web nativas.
